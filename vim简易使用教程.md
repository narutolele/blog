# 什么是vim #
Vim?是?Linux 系统上的最著名的文本/代码[编辑器]，也是早年的 Vi 编辑器的加强版，而 gVim 则是其 Windows 版。它的最大特色是完全使用键盘命令进行编辑，脱离了鼠标操作虽然使得入门变得困难，但上手之后键盘流的各种巧妙组合操作却能带来极为大幅的效率提升。

#为什么要学习vim#
- 因为作为程序员，是一定要和Linux打交道的，而vim正是linux系统下的文本编辑器。
- 现有的服务器大多数为linux系统。
- 修改某个服务器中的配置文件时，vim的作用就会体现出来。

#vim简易使用方法#
vim编辑器分为两种模式，一种为命令模式，一种为编辑模式，当你通过vim 文件名.后缀名（注意！vim和文件名之间有空格）进入vim之后默认为命令模式，在命令模式中是不能编辑的，可以通过键盘上的字母i来进入编辑模式，，进入编辑模式后左下角会出现INSERT，这时就可以编辑文本了，退出编辑模式为esc。

- dd 删除当前行
- wq 保存退出
- q! 不保存退出
- dd 删除当前行
- Ctrl + v 在当前光标处开始选文字（相当于windows下按住鼠标拖动选中）
- y复制
- / + 内容  在光标后查找
- ? + 内容   在光标前查找
